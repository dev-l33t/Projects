🎨 UI Components Testing

⬇️ Dropdown Menus

Test opening, closing, selecting an option, and ensuring accessibility via keyboard navigation.

⏳ Loaders and Spinners

Validate appearance during page/data loads and ensure loaders disappear after completion.

📊 Progress Bars

Test progress bar movement, percentage updates, and completion states visually and functionally.

📈 Charts and Graphs

Verify correct data rendering in charts (bar, pie, line) and responsive resizing behavior.

🖼️ Modal Popups

Test opening, closing modals, backdrop clicks, and escape key functionality.

🍞 Toast Notifications

Test auto-hiding notification alerts, appearance timing, and click-to-dismiss actions.

💬 Tooltips

Test tooltip visibility on hover/focus and ensure correct positioning and timing delays.

📚 Accordion Panels

Test expanding and collapsing multiple accordion sections and maintaining their state.

🖼️ Image Carousels

Test carousel sliding, next/previous controls, auto-play, and manual interactions.

📄 Pagination Controls

Test navigating between pages of content, disabling previous/next buttons at limits.

🎚️ Sliders and Range Inputs

Test adjusting values using sliders, min/max limits, and real-time feedback updates.
