🖱️ Interactions Testing

🧲 Drag and Drop

Test dragging elements and dropping them into designated targets. Validate correct movement and visual feedback.

🖱️ Double Click

Test double-click actions on elements. Verify that double-clicking triggers the intended event and response.

📋 Right-Click Context Menu

Test right-click (context) menus, menu options visibility, and actions on selection.

🎯 Hover Effects

Test mouse hover interactions, tooltip visibility, and style changes when hovering over elements.

✔️ Selectable Items

Test selecting multiple list/grid items using click, Ctrl+Click, or Shift+Click combinations.

🔀 Sortable List

Test dragging to reorder list items and validating the new order after sorting.

↔️ Resizable Elements

Test resizing elements by dragging corners/edges. Validate min/max size restrictions and content adjustments.

⌨️ Keyboard Events

Test keypress, keydown, and keyup events. Validate form submissions using Enter, shortcuts using Ctrl, and tab navigation.

🖲️ Mouse Movement Tracking

Test capturing and reacting to mouse movements, pointer positions, and mouseover highlights.

📜 Scroll Events

Test page and element scroll events, sticky elements behavior, and infinite scrolling triggers.
